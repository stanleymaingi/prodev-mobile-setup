# Mobile Development Environment Setup

## Objective

Set up a mobile development environment using Expo Framework for React Native.

---

## 🛠 Tools Installed

- Node.js (LTS) – v18.x.x
- npm – v9.x.x
- VS Code
- Operating System: Windows 10
- Expo Go (installed on Android device)

---

## 📱 Expo Go Installation Process

1. Visited https://expo.dev/go
2. Selected latest SDK version.
3. Installed Expo Go from Google Play Store.
4. Opened the app.
5. Created an Expo account.
6. Logged in successfully.

---

## 🧪 Testing the Setup

1. Created a test Expo project using:
   npx create-expo-app test-app

2. Started the development server:
   npm start

3. Scanned the QR code using Expo Go.

4. Successfully ran the test application on my physical device.

---

## ⚠️ Challenges Faced

- QR code not scanning initially.
- Resolved by ensuring both devices were on the same WiFi network.

---

## ✅ Conclusion

The mobile development environment is successfully configured and ready for React Native development using Expo.
